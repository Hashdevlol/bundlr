Simple

Three commands. That's it.

No complex setup. No configuration hell. Install, initialize, launch. The CLI handles everything else — wallet generation, funding, coordination, and profit collection.

# Install the CLI
$ npm install -g bundlr-cli
# Initialize your project
$ bundlr init
# Launch your token
$ bundlr launch
