# Bundlr

**Launch tokens like a pro.** One-click bundle launcher for Solana.

## Features

- **20-Wallet Bundler** — Generate, fund, and coordinate wallets automatically
- **Staggered Buys** — 3 groups buying at different times to simulate organic activity
- **Cascading Sells** — Automated profit-taking with configurable thresholds
- **Website Generator** — Launch-ready landing page in 30 seconds
- **Timing Optimizer** — Data-driven insights on best launch windows

## Quick Start

```bash
npm install -g bundlr-cli
bundlr init
bundlr launch
```

## How It Works

1. **Generate** — Create 20 fresh wallets in 3 groups
2. **Fund** — Distribute SOL from your main wallet
3. **Launch** — Deploy token + coordinate buys
4. **Profit** — Cascading sells as price rises
5. **Collect** — Sweep profits back to main wallet

## Stack

- Solana/web3.js
- Helius RPC
- Next.js (dashboard)
